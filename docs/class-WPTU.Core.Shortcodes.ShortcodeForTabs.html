Class ShortcodeForTabs

Manages tabs design with shortcodes.

Use [tab]<some text>[/tab] to wrap the content of a single tag and then [tabs-list] for the list of the tabs in the current post content.

These optional parameter are available: icon = string|int number if you want a incon from the media library, string for the <img> tag src attribute class = string class attribute for the current tab div title = string the title of the tab list = boolean set to false if you want to remove the current entry from the tabs list from = the starting point of the google maps route route-type = the type or google maps route google.maps.TravelMode
