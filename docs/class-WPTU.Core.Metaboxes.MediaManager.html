Class MediaManager

Manages multiple media set

This feature adds a metabox to your WordPress admin panel in which you can add any media media into a given set.

Then in your theme you will be able to retrieve the single set and show it where you need it.

For example if you need to have a slideshow on the top of the page and a minigallery on the bottom, this feature is what you need to separate your images between the two sets.

In your functions.php simply add

MediaManager::enable();
MediaManager::set_media_list('slideshow', array(
        'label'         =>      __('Slideshow', 'wtu_framework'),
        'shortcode'     =>      'mySlideshow'
));

to enable the backend feature.

Now you will have to manage a new shortcode; for example:

function mySlideshow_gallery_shortcode($atts){
        $data = shortcode_atts(array('ids' => ''), $atts );
        $toret = '';
        $ids = explode(',', $data['ids']);
        foreach($ids as $id){
                $toret .= wp_get_attachment_image($id);
        }
        return $toret;
}

add_shortcode('mySlideshow', 'mySlideshow_gallery_shortcode');

If you need more than a section simply duplicate your code and change at least the shortcode and the id (first parameter of set_media_list())
