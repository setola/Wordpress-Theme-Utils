Classes summary

BootstrapCarousel Manages the markup for a generic set of images Extend this with your nice and candy classes!
ImagePreload Takes care of image preloading with jQuery imagesloaded
ImageProgressiveLoading Takes care of image sequential loading with jQuery Cycle 2
MergedSlideshow Merge all the images into a big one and serve it as div background
MinigalleryBigImageWithThumbs Manages the markup for the minigallery with a big image and some small thumbs
MinigalleryThumbsLinkToBig Generates a list of thumbnail each one is an anchor to the image with 'full' media dimensions. By default loads FancyBox so to have an eye candy popup effect.
OneImageForAll Merges all images of a gallery in only one and serve it as a sprite Important: every image have the same sizes and jpeg quality.
PhotogalleryThumbWithFancybox Manages the markup for a generic set of images Extend this with your nice and candy classes!
